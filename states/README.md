## How to read each state page

Each row is one moratorium adopted by one local government (a city, county, township, etc.). "Date enacted" is when the local board voted; "duration" is how long the original moratorium was scheduled to run; "sector" is what kind of project it pauses (data centers, crypto mining, batteries, solar, wind).
